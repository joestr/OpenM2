1. Schritt

Ware wählen:

Wählen Sie in der Ablage Warenangebot den entsprechenden Katalog und aus dem Katalog die entsprechende Ware aus

 

 

Suche unter:

Wie suche ich Waren

2. Schritt

Ware merken:

Sie bekommen in der Infoansicht schon die Möglichkeit, die Ware für die Bestellung vorzumerken, indem Sie auf das Symbol „Merken“ neben dem Bild der Ware klicken.

 

Indem Sie auf die gewünschte Ware klicken, erhalten Sie eine detaillierte Beschreibung der Ware. Die Ware wird für die Bestellung vorgemerkt, indem Sie auf den Button „Merken“ klicken.

 

Es öffnet sich ein Fenster, in dem Sie das Bestellformular für die Ware ausfüllen können.

 

 

 

3. Schritt

Ware in Warenkorb legen:

Sie können nun die gewünschte Menge der ausgesuchten Ware eingeben.

 

Wurde ein Sortiment angelegt, können Sie nur das gesamte Sortiment bestellen.

 

Nach dem Klick auf den „OK“ Button, öffnet sich ein Dialogfenster „Das Produkt wurde in ihren Warenkorb gelegt“.

 

 

Suche unter:

Sortiment anlegen

Warenkorb

Privatansicht

4. Schritt

Ware Bestellen:

Sie wechseln nun in die Privatansicht in die Ablage Warenkorb. Dort sehen Sie die zur Bestellung vorgemerkten Waren. Klicken Sie nun auf den Button „Bestellen“.

 

Die Bestelliste zeigt die Kontaktperson für die Bestellung an, den Lieferanten, den Händler, die Kontaktperson beim Händler und die Liste der ausgewählten Waren inklusive den Preisen und dem Gesamtpreis der Bestellung an. Sie haben die Möglichkeit, die Bezeichnung der Bestellung, die Bestellnummer, die Lieferadresse, die Rechnungsadresse, die gewünschte Vorgangsweise, wenn die Ware nicht lagernd ist, die gewünschte Lieferform, eine Bestellbemerkung und das gewünschte Lieferdatum einzugeben.

 

Nach dem Klick auf den „OK“ Button erhalten Sie vom System die Benachrichtigung, daß die Bestellung durchgeführt wurde.
