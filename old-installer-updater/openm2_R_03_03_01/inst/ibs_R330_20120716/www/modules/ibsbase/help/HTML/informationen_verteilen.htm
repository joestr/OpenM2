1. Schritt

Ablage wählen:

Sie wählen unter den Ablagen diejenige aus, aus der Sie eine Information verteilen möchte

 

Suche unter:

Wie ist das System aufgebaut

2. Schritt

Information wählen:

 

Durch den Klick auf die Ablage, sehen Sie die Informationen, die sich in dieser Ablage befinden

 

 

3. Schritt

Information

Verteilen:

Klicken Sie nun auf die entsprechende Information, bekommen Sie in der Infoansicht den Button „Verteilen“ angezeigt.

 

Nach dem Klick auf den Button öffnet sich ein Dialogfenster mit Auswahlmöglichkeiten. Sie haben nun die Möglichkeit, Filter zu setzen, je nachdem, welche Gruppen bzw. Benutzer diese Information in den privaten Eingangskorb bekommen sollen.

 

Als Betreff wird der Name der verteilten Information automatisch eingefügt.

 

Damit die Information nicht mehr bearbeitet werden kann, hat der Ersteller der Information die Möglichkeit, die Information für weitere Bearbeitung zu sperren.

 

Mit der verteilten Information können Sie auch eine Anforderung mitschicken, was mit der Information geschehen soll. Sie können beispielsweise um Durchsicht bzw. Bearbeitung bitten.

 

Zusätzlich können von Ihnen noch Anmerkungen zu der verteilten Information gemacht werden.

 

Nach der Bestätigung der Verteilung erhalten Sie die Nachricht, daß die Information erfolgreich verteilt wurde.

 

 

Suche unter:

Was ist die Privatansicht

4. Schritt

Übermittlung der Information

Die Information wird in den privaten Eingangskorb jener Benutzer verteilt, die vom Eigentümer der Information zur Verteilung ausgewählt wurden.

 

Wenn Sie die Information verteilt haben, so gelangen Sie in Ihren Ausgangskorb
