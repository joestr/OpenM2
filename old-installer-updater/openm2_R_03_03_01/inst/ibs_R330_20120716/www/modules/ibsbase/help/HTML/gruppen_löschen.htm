Um Gruppen aus dem System zu entfernen, haben Sie 2 Möglichkeiten.

 

1. Schritt

Ablage wählen

Wählen Sie die Ablage Benutzerverwaltung und klicken Sie dort auf „Gruppen“

 

 

Siehe auch:

Wie suche ich eine Information

 

2. Schritt

Gruppe wählen

Sie erhalten eine Liste aller Gruppen im System. Klicken Sie nun auf den Button „Liste löschen“.

 

 

 

3. Schritt

Gruppe löschen

Klicken Sie entweder auf das Feld neben der Gruppe, die sie löschen möchten, oder wählen Sie eine Option aus den Buttons aus. Die Optionen wie „Alles Auswählen“, „Auswahl aufheben“ und „Auswahl umdrehen“ sind vorgegeben.

 

Nachdem Sie die Gruppe markiert haben, die Sie löschen möchten, klicken Sie auf „OK“.

 

Sie erhalten nun wieder eine Liste aller Gruppen des Systems, jedoch ohne die gelöschte Gruppe.

 

 

 

Hinweis:

Es werden alle Rechteprofile dieser Gruppe gelöscht

 

Löschen einer abgelaufenen Gruppe

Um einen Gruppe aus dem System zu löschen, deren „Verfallsdatum“ abgelaufen ist, klicken sie in der Privatansicht im Benutzerprofil auf den Button „Säubern“. Sie erhalten hier eine Liste mit allen abgelaufenen Objekten.
