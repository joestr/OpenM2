1. Schritt

Ablage wählen:

Sie wählen unter den Ablagen diejenige aus, aus der Sie eine Information mit einem Link versehen möchte.

 

Suche unter:

Wie ist das System aufgebaut

2. Schritt

Information wählen:

 

Durch den Klick auf die Ablage, sehen Sie die Informationen, die sich in dieser Ablage befinden

 

 

3. Schritt

Link erstellen:

Klicken Sie nun auf die entsprechende Information, bekommen Sie in der Infoansicht den Button „Kopieren“ angezeigt.

 

Nach dem Klick auf den Button öffnet sich ein Dialogfenster „Objekt zum kopieren markiert“.

 

 

Suche unter:

Wie kopiere ich Informationen

 

4. Schritt

Link einfügen:

Sie haben nun die Möglichkeit, die Information als Link in eine andere Ablage einzufügen, ohne die ganze Information kopieren zu müssen. Das Einfügen erfolgt über den Button „Link einfügen“ in der Infoansicht der Ablage.

 

Ob eine Information mit einem Link versehen ist, sehen Sie im Verweise-Reiter.

 

Suche unter:

Einführung in das Basissystem

Reiter
