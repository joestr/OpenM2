1. Schritt

Ablage wählen

Wählen Sie die Ablage Benutzerverwaltung und klicken Sie dort auf „Benutzer“

 

 

Suche unter:

Wie suche ich eine Information

 

2. Schritt

Benutzer wählen

Sie erhalten eine Liste aller Benutzer im System. Klicken Sie nun auf den Button „Liste löschen“.

 

 

 

3. Schritt

Benutzer löschen

Klicken Sie entweder auf das Feld neben dem Benutzer, den sie löschen möchten, oder wählen Sie eine Option aus den Buttons aus. Die Optionen „Alles Auswählen“, „Auswahl aufheben“ und „Auswahl umdrehen“ sind vorgegeben.

 

Nachdem Sie die Benutzer markiert haben, die Sie löschen möchten, klicken Sie auf „OK“.

 

Sie erhalten nun wieder eine Liste aller Benutzer des Systems, jedoch ohne die gelöschten Benutzer.

 

 

 

 

 

Löschen eines abgelaufenen Benutzers

Um einen Benutzer aus dem System zu löschen, dessen „Verfallsdatum“ abgelaufen ist, klicken sie in der Privatansicht im Benutzerprofil auf den Button „Säubern“. Sie erhalten hier eine Liste mit allen abgelaufenen Objekten.

 

Suche unter:

Was ist die Privatansicht

Benutzerprofil

 

Löschen über die Suchfunktion

Sie können einen Benutzer auch über die Suchfunktion löschen. Geben Sie den Namen des Benutzers in die Suchmaske ein, den Sie löschen möchten. Bei der Ergebnisliste der Suche erhalten Sie den Button „Liste löschen“. So können Sie direkt über die Suche den Benutzer aus dem System löschen.

 

Suche unter:

Wie lösche ich Informationen
