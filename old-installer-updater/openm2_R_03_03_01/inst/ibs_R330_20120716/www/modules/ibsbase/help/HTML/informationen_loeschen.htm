1. Schritt

Ablage wählen:

Sie wählen unter den Ablagen diejenige aus, aus der Sie eine Information löschen möchte.

 

Suche unter:

Wie ist das System aufgebaut

2. Schritt

Information wählen:

 

Durch den Klick auf die Ablage, sehen Sie die Informationen, die sich in dieser Ablage befinden

 

 

3. Schritt

Information löschen:

In der Infoansicht bekommen Sie den Button „Liste löschen“. Klicken Sie auf diesen Button, erhalten Sie die vollständige Liste aller Informationen, die sich in dieser Ablage befinden.

 

Sie können nun durch anklicken auswählen, welche Information gelöscht werden soll. Eine andere Möglichkeit Informationen zu löschen ist durch die Auswahl von vordefinierten Feldern gegeben.

 

Kreuzen Sie das Feld „Alles auswählen“ an, werden alle in der Ablage befindlichen Informationen zum Löschen markiert. Mit „Auswahl aufheben“ wird die Markierung der Informationen wieder aufgehoben. Über das Feld „Auswahl umdrehen“ werden alle markierten Informationen zu unmarkierten und umgekehrt.

 

 

 

Löschen von abgelaufenen Informationen

 

Befinden sich im System Informationen, deren „Verfallsdatum“ überschritten wurde, können diese über den Button „Säubern“ in der Privatansicht im Benutzerprofil gelöscht werden.

 

 

Suche unter:

Was ist die Privatansicht

Benutzerprofil

 

 

Löschen von Informationen nach der Suche

 

Nachdem Sie das Suchergebnis in einer Liste erhalten, haben Sie die Möglichkeit, die gefunden Objekte sofort über den „Liste löschen“ Button zu entfernen. Sie können damit direkt nach Informationen suchen, die Sie sofort aus dem System löschen möchten.

 

Suche unter:

Wie suche ich eine Information

Wie lösche ich eine Information
