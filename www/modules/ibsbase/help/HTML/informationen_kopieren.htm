1. Schritt

Ablage wählen:

Sie wählen unter den Ablagen diejenige aus, aus der Sie eine Information kopieren möchte

 

Suche unter:

Wie ist das System aufgebaut

2. Schritt

Information wählen:

 

Durch den Klick auf die Ablage, sehen Sie die Informationen, die sich in dieser Ablage befinden

 

 

 

3. Schritt

Information kopieren:

Klicken Sie nun auf die entsprechende Information, bekommen Sie in der Infoansicht den Button „Kopieren“ angezeigt.

 

Nach dem Klick auf den Button öffnet sich ein Dialogfenster „Objekt zum kopieren markiert“.

 

 

 

4. Schritt

Information in andere Ablage einfügen

Sie navigieren nun in die gewünschte Ablage und fügen dort die Information über „Einfügen“ ein.

 

 

 

Kopieren einer „fremden“ Information:

Sie können Informationen, auch wenn sie nicht von Ihnen eingebracht wurden, kopieren. Dazu sind die Schritte 1 – 3 von Informationen kopieren zu wiederholen. Allerdings besteht in diesem Fall nur die Möglichkeit, die kopierte Information in den Privaten Arbeitskorb als Link oder als Informations-Objekt einzufügen.

 

Suche unter:

Wie erstelle ich einen Link auf eine Information
