1. Schritt

Ablage wählen :

Sie wählen unter den Ablagen diejenige aus, in der Sie eine neue Information erstellen möchten. Neue Informationen sind in jeder Ablage möglich. Es hängt dabei von der Ablage ab, welche Informationstypen zugelassen werden.

 

Das erstellen einer neuen Information wird im folgenden anhand des Beispiels der „Notiz“ erklärt.

 

 

Suche unter:

Wie ist das System aufgebaut

2. Schritt

Neue Information anlegen:

Haben Sie die entsprechende Ablage angewählt, klicken Sie nun auf den Button „Neu“, um eine neue Information erstellen zu können.

 

Es erscheint die Eingabemaske, über die Sie die gewünschte Information eingeben können. Zuerst wird der Name eingegeben, den die neue Information erhalten soll

 

 

 

3. Schritt

Typ wählen:

Als nächstes wählen Sie den Typ von Information, den Sie erstellen möchte. Dabei ist darauf zu achten, daß nicht in jeder Ablage jeder Typ von Information möglich ist. Es erscheint in jeder Ablage beim Typ eine Vorauswahl, welche Informationsarten für die jeweilige Ablage zugelassen sind.

 

Möchten Sie eine Notiz erstellen, so geben Sie beim Typ „Notiz“ ein.

 

Suche unter:

Wie ist das System aufgebaut

 

Hinweis:

Nicht jeder Typ von Information ist in allen Ablagen möglich.

 

4. Schritt

Beschreibung:

Im Feld „Beschreibung“ geben Sie eine genaue Bezeichnung der Information ein.

 

Bei der Erstellung einer Notiz wird hier die genauere Beschreibung der Notiz eingetragen.

 

Anschließend klicken Sie auf den „OK“ Button.

 

 

 

5. Schritt

Gültigkeit:

Jede Information hat ein „Verfallsdatum“. Hier können Sie eingeben, wie lange die Information im System erhalten bleiben soll

.

 

 

6. Schritt

Inhalt

Der Inhalt der Information besteht aus der Beschreibung und der Zusatzinformation, die von Ihnen eingegeben werden kann.

 

 

 

7. Schritt

Nächste Aktion wählen:

Sie haben nun die Möglichkeit, beim Abspeichern des Objektes schon die nächstfolgende Aktion „anzumelden“.

 

Möchten Sie nach der Speicherung der Information wieder in die Ablage kommen, markieren Sie das Feld „Speichern und zur Ablage“. Möchten Sie zu dem neuen Objekt selbst, wählen Sie „Speichern und zum Objekt“. Möchten Sie ein weiteres Objekt erstellen, wird das Feld „Speichern und Neu“ markiert. Wird das Feld „Zur Rechteverwaltung“ angeklickt, kommen Sie zur Rechtevergabe auf das Objekt.
