1. Schritt

Ablage wählen:

Sie gehen in die Ablage, aus der Sie eine Zusatzinformation speichern möchte.

 

Suche unter:

Wie lese ich Informationen

2. Schritt

Information wählen:

In der Ablage erhalten Sie eine Liste, der dort zur Verfügung stehenden Informationen. Befindet sich neben dem Namen einer Information ein Info-Icon, so gibt es zu dieser Information eine Zusatzinformation, die gespeichert werden kann.

 

 

 

3. Schritt

Speichern :

Die Datei wird entsprechend den Einstellungen im Benutzerprofil angezeigt.

 

Sie können Dateien mit den Endungen *.doc und *.xls speichern, indem Sie den Browser dazu benützen. Das heißt, daß Sie direkt über das Browsermenü „Datei“ und „Speichern unter“ die Datei abspeichern können.

 

Dateien mit der Endung *.pdf können direkt im Acrobat-Reader unter dem Button „Speichern“ (Icon Diskette) abgespeichert werden.

 

Suche unter:

Was ist die Privatansicht
