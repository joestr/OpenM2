1. Schritt

Ablage wählen

Wählen Sie die Ablage Benutzerverwaltung und klicken Sie dort auf „Benutzer“

 

 

Suche unter:

Wie suche ich eine Information

 

2. Schritt

Benutzer wählen

Klicken Sie auf denjenigen Benutzer, den Sie einer neuen Gruppe zuordnen möchten. Aus der Info-Sicht des Benutzers klicken Sie nun auf den Reiter „Mitglied in“. Sie erhalten die Liste, in welchen Gruppen der Benutzer Mitglied ist.

 

 

 

3. Schritt

Gruppe wählen

Klicken Sie nun auf den Button „Zuordnen“, um den Benutzer zu bestehenden Gruppen zuordnen zu können.

 

Jeder Benutzer wird vom System automatisch zur Gruppe „Jeder zugeordnet“.

 

Um die Zuordnung durchzuführen, haben Sie die Möglichkeit, die Felder vor dem Gruppennamen anzuklicken, oder über die Buttons „Alles Auswählen“, „Auswahl aufheben“ oder „Auswahl umdrehen“ den Benutzer zuzuordnen.

 

Indem Sie nach der Zuordnung auf den Button „OK“ klicken, sehen Sie, in welchen Gruppen der Benutzer nun Mitglied ist.

 

 

Suche unter:

Wie lege ich neue Gruppen an.
