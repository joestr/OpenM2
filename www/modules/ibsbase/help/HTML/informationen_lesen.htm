1. Schritt

Ablage wählen:

Die Ablagen im Gruppenbaum beinhalten alle Informationen des Systems. Durch einen Klick auf einen Namen einer Ablage erhalten Sie eine Liste aller Informationen, die in dieser Ablage sind. Über die verschiedenen Reiter (wie beispielsweise Info, Verweise, etc.) können Sie zwischen den verschiedenen Sichten einer Information wechseln.

 

 

Suche unter:

Wie ist das System aufgebaut

 

2. Schritt

Einstellungen zur Information:

Im Benutzerprofil haben Sie die Möglichkeit, die Einstellung vorzunehmen, ob Sie erweiterte Informationen erhalten möchten. Stellen Sie die erweiterte Informationsansicht ein, so sehen Sie in der Spaltenansicht der Information nicht nur den Namen und den Typ der Information, sondern auch den Eigentümer und „geändert am“. Diese Einstellung gilt für alle Informationen des gesamten Systems.

 

Bei der Einstellung für die Zusatzinformation haben Sie auch die Wahlmöglichkeit, ob Sie die Zusatzinformationen in einem eigenen Browser-Fenster sehen möchte. Wird diese Funktion eingestellt, so öffnet sich beim Klick auf das Info-Icon ein eigenes Fenster, in dem die Zusatzinformation dargestellt wird.

3. Schritt

Zusatz-informationen

Existiert zu einer bestimmten Information eine Zusatzinformation, können Sie dies durch das Info-Icon neben dem Namen der Information erkennen. Klicken Sie auf den Namen, wird die Information geladen. Klicken Sie auf das entsprechende Info-Icon erhalten Sie die entsprechende Zusatzinformation. In der Zusatzinformation können Dateien liegen, die mit einem Viewer angesehen werden können. Mögliche Dateiformate sind beispielsweise *.doc, *.pdf *.txt.

 

 

 

4. Schritt

Informationen sortieren:

Die Liste der Informationen kann nach allen beliebigen Attributen der Information sortiert werden. Die Attribute bilden auch die Spalten der Informationsliste. Das Sortieren erfolgt über den Klick auf den Namen der Spalte. Je nachdem, nach welchem Attribut sortiert wurde, erscheint dann ein kleines Dreieck neben dem Namen der Spalte. Bei der Sortierung haben Sie auch die Möglichkeit, die Informationen auf- bzw. absteigend zu sortieren.
